        【例题1】字符串哈希 的另一种做法

        Zxgaer小于 1 分钟set算法

        标答用的Hash，但我觉得STL厉害，我用set

        #include<iostream>
#include<set>
using namespace std;
int n;
string a;
set<string> s;
int main() {
    scanf("%d",&n);
    while(n--) {cin>>a;s.insert(a);}
    cout << s.size();
}
